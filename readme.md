# Brief
Library for communicating ADNS3080 and stm32. Read basic values from ADNS380 optical flow sensor.
Tested on STM32F407VG board.
# Pinout.
The pinout between STM32 and ADNS3080 is as follow:
+ ADNS3080 -> STM32
+ Power pin.
+ NCS -> PE7
+ MISO -> PB6
+ MOSI -> PB7
+ SCLK -> PA5
+ RST -> PB1
